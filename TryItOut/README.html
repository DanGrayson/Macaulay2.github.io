Macaulay2 server

You can try out Macaulay2 in a web browser at Macaulay2 Server. The web application running there was written by Franziska Hinkelmann, Lars Kastner, and Mike Stillman. Read the "Welcome Tutorial" there to find out how to interact with Macaulay2.

Macaulay2Web Server -- new

You can try out Macaulay2 in a web browser at Macaulay2Web. The web application running there was written by Paul Zinn-Justin, based on the one above by Franziska Hinkelmann, Lars Kastner, and Mike Stillman, and enhanced to display the output as though it had been typeset by TeX. Read the "Welcome Tutorial" there to find out how to interact with Macaulay2.

CoCalc

You can also try out Macaulay2 in a web browser on the CoCalc, thanks to William Stein. Create an account, log in, create a new project, create a new terminal, and then run "M2". Refer to Getting Started for ideas about what commands to try out.

SageMathCell

You can also try out Macaulay2 in a web browser in SageMathCell, thanks to Andrey Novoseltsev.
