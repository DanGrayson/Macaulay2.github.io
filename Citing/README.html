If you cite Macaulay2 in a paper, please send us a reference, and we'll add it to our list of publications.

If you like, you may use one of the bibtex entries below for citing Macaulay2. The first version is for use with the hyperref macro package.

Note: in October, 2009, we decided to refer uniformly to Macaulay2 (with no internal space) instead of to Macaulay 2. 

        @Misc{M2,
          author = {Grayson, Daniel R. and Stillman, Michael E.},
          title = {Macaulay2, a software system for research 
                   in algebraic geometry},
          howpublished = {Available at 
              \href{http://www.math.uiuc.edu/Macaulay2/}%
                   {http://www.math.uiuc.edu/Macaulay2/}}
        }
